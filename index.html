GN4R is mostly written in Java/Kotlin and his code is maintained on GitHub.
Interested in the source code or contributing? Click here to go to the GitHub page.

GN4R is a side-project written by Avalon and Maeyrl.

About GNAR

GN4R is a Discord bot written by Avalon and Maeyrl.

Fun Commands

GNAR comes packed with fun and silly commands, such as [_zalgo] and [_eyes], along with game info lookup for Overwatch and Leauge of Legends.

Comics and Media

We got comics such as xkcd, Cyanide and Happiness, Garfield to warm up your soul, alongside with several media commands like cats, Google and YouTube.

Private Messages

Want to chat with GNAR or use a command in private? Use private messages with him! Call a command like [_help] or [_info], or just start typing to chat with an AI.
