GN4R
The Sarcastically Serious Bot


Commands

GN4R is a seriously sarcastic or sarcastically serious Discord bot, but who really knows?

Looking for the commands? Check the side-bar or type [ _help ] on a server with GN4R.

Invite GN4R

Need GN4R to wake your soul from the darkest deepest depths of Tartarus?

Invite him to your server today, and enjoy some GN4R goodness.

Open Source

GN4R is mostly written in Java/Kotlin and his code is maintained on GitHub.
